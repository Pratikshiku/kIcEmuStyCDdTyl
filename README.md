## Teacher Personnel File Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/2177c6c1647c482db7ee36f2b1079211/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User (Teacher), Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Training Information (Favorite Courses, Comment on Courses, Register for Courses)
5. Forum Information (Publish Forum Posts, Comment on Others' Posts)
6. System Announcements
7. Personal Center, Modify Information and Password
8. My Posts
9. My Favorites
10. Customer Service Chat
11. My Personal Profile
12. Profile Changes
13. My Reward and Punishment Information
14. My Training Registration
15. My Course Information
16. My Favorites Management


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Modify Information)
3. Teacher Management
4. Personal Profile Management
5. Profile Change Management
6. Reward and Punishment Information Management
7. Training Information Management
8. Training Registration Management
9. Course Information Management
10. Forum Management
11. System Announcement Management
12. Customer Service Management
13. Carousel Management
